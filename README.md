如何安装
------------

### 下载依赖包

#### 1.增加仓库
``` gradle
allprojects {
	repositories {
		...
		maven { url 'https://jitpack.io' }
	}
}
```
#### 2.增加依赖
```gralde
dependencies {
    compile 'com.github.tevid:jbox:0.0.1'
}
```


*PS*: 最好使用 jdk 1.8+
